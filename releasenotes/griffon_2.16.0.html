Griffon 2.16.0 Release Notes

16 December 2021

Build

The build has been upgraded to Gradle 6.9 and relies on the Kordamp Gradle plugins.

It’s no longer possible to use Lazybones to bootstrap projects. Instead, use the provided Maven archetypes. The guide has updated information on how they may be used.

Compatibility

A list of changes can be found at the v2.16.0 release page.
